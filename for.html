Glosario de Funciones

cargar_imagen(ruta)
Carga una imagen desde la ruta del sistema de archivos especificada. Retorna un objeto de imagen de OpenCV o None si falla.
mostrar_imagen(imagen, titulo)
Despliega la imagen en una ventana con el título especificado. La ventana permanece abierta hasta que se presione una tecla.
obtener_dimensiones(imagen)
Extrae y retorna las dimensiones de la imagen como tupla: altura, ancho y número de canales de color.
cv2.imread()
Función de OpenCV que lee una imagen desde un archivo. Soporta múltiples formatos como JPG, PNG, BMP, etc.
cv2.imshow()
Función de OpenCV que muestra una imagen en una ventana nueva. Requiere cv2.waitKey() para mantener la ventana abierta.
cv2.destroyAllWindows()
Cierra todas las ventanas de OpenCV que están abiertas. Útil para liberar recursos después de mostrar imágenes.
